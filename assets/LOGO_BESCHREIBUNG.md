# PflegeNavigator EU - Logo-Paket

## Überblick
Modernes, professionelles Logo-Paket für Print und Digital.

### Design-Philosophie
- **Kompass**: Orientierung, Navigation, Wegweiser
- **Herz-Element**: Fürsorge, Menschlichkeit, Pflege
- **Nadel**: Dynamisch, zeigt Richtung (N wie Navigator)
- **Farben**: 
  - #0f2744 (Dunkelblau): Vertrauen, Professionalität
  - #20b2aa (Türkis): Frische, Innovation, Pflege
  - Weiß: Klarheit, Einfachheit

## Für Briefpapier (Druck)

### Empfohlene Einstellungen:
- **Format**: logo-horizontal.svg oder logo-pflegenavigator.svg
- **Auflösung**: Mindestens 300 DPI (SVG skaliert verlustfrei)
- **Farben**: CMYK-Konvertierung für professionellen Druck
- **Position**: Links oben, 2,5cm vom Rand
- **Größe**: Höhe 2-3cm je nach Briefkopf-Design

### Konvertierung zu anderen Formaten:

```bash
# SVG zu PNG (hochauflösend)
# Nutze: inkscape, illustrator, oder online: cloudconvert.com

# Für Word/Briefpapier:
# 1. SVG in Word einfügen (moderne Word-Versionen unterstützen SVG)
# 2. Oder: Konvertiere zu PNG mit 300 DPI
# 3. Oder: Konvertiere zu PDF für Vektor-Qualität
```
